## 目录说明


| 目录                   | 说明                            |
| -------------------- | ----------------------------- |
| space                | 包含了所有的笔记文件                    |
| resource             | 一些可以复用的资源文件                   |
| resource/attachments | 附件文件夹                         |
| resource/components  | 组件文件夹（包含了所有的 components 组件文件） |
| resource/images      | 图片素材文件夹                       |
| resource/template    | 模板文件夹                         |
